--- Storage Health Observations ---
The root filesystem shows adequate free space for current usage.

--- Recommendations for Improving Storage Utilization ---
1. Set up monitoring alerts when disk usage crosses 80% threshold.
2. Regularly clean up temporary files and old logs to reclaim space.
3. Consider separate partitions/volumes for logs vs application data
   so a runaway log file cannot fill the entire root filesystem.
4. Monitor inode usage separately from disk space, especially if the
   application creates many small files (e.g., session files, cache).
5. Use LVM or similar for flexible storage resizing as the application
   server's needs grow over time.
